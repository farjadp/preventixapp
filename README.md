# Preventix Website

Official website for Preventix - a Canadian health-tech company focused on preventive dental health through environmental design.

## About

Preventix reduces dental risk by changing food environments, not human behavior. We work with employers, insurers, and institutions to create measurable impact through environmental infrastructure.

## Website Structure

The website consists of the following pages:

- **index.html** - Homepage with hero section and key pillars
- **how-it-works.html** - Step-by-step explanation of the Preventix system
- **product.html** - Product roadmap and phased approach
- **who-its-for.html** - Target segments (employers, insurers, institutions)
- **science.html** - Evidence-based approach and clinical oversight
- **privacy.html** - Privacy commitments and PIPEDA compliance
- **team.html** - Team information (requires customization)
- **contact.html** - Contact form for pilot partnerships

## Design Philosophy

- Clean, minimal, professional aesthetic
- White background with soft blue/muted green accents
- No gradients or heavy animations
- Focus on trust and credibility
- Built for investors, corporate partners, and insurance providers

## To Deploy

Simply host the files on any static web server. The website is pure HTML/CSS with no build process required.

## Customization Needed

Before going live, update:

1. Team member information in `team.html`
2. Contact form action URL in `contact.html`
3. Add actual team photos/headshots if desired

## Technology

- Pure HTML5 and CSS3
- Responsive design
- No JavaScript dependencies
- Mobile-friendly
